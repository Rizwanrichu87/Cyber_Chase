🔍 HOW TO SOLVE THE PASSWORDS
═══════════════════════════════════════════════════════════════════════════════

LEVEL 1 SOLUTION EXAMPLE:
Question: "What protocol is used for secure shell access? Add 47 to it."
Thought process:
  1. What protocol for secure shell? → SSH
  2. What do we add? → 47 (minutes in the breach time 02:47)
  3. Format: The word + number → PROTOCOL47
Answer: PROTOCOL47 ✓

LEVEL 2 SOLUTION EXAMPLE:
Question: "What is Microsoft's enterprise email software? Add the year."
Thought process:
  1. Microsoft's email server? → Exchange
  2. Current year? → 2024
  3. Format: Software name + year → EXCHANGE2024
Answer: EXCHANGE2024 ✓

LEVEL 3 SOLUTION EXAMPLE:
Question: "What is the OS kernel? Described by what architecture?"
Thought process:
  1. Core OS software? → Kernel
  2. Modern architecture? → 64-bit (most common today)
  3. Format: Term + architecture → KERNEL64BIT
Answer: KERNEL64BIT ✓

LEVEL 4 SOLUTION EXAMPLE:
Question: "Network protocol suite? Secure port number? Encryption?"
Thought process:
  1. Network foundation? → TCP/IP
  2. Secure HTTPS port? → 443
  3. Encryption standard? → SSL (or TLS)
  4. Format: Combine all → TCPIP443SSL (no spaces or slashes)
Answer: TCPIP443SSL ✓

LEVEL 5 SOLUTION EXAMPLE:
Question: "Extract and combine elements from previous passwords"
Thought process:
  1. From PROTOCOL47: First 4 letters → PROT
  2. From EXCHANGE2024: Last 4 letters → 2024
  3. From KERNEL64BIT: First letter (K) + digit (6) → K6
  4. From TCPIP443SSL: Middle letter (P) + port digit (3) → P3
  5. Combine in order: PROT + 2024 + K6 + P3 → PROT2024K6P3
Answer: PROT2024K6P3 ✓
