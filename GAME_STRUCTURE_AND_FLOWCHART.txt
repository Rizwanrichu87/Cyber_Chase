💡 HINT PROGRESSION SYSTEM (ADMIN REQUIRED)
═══════════════════════════════════════════════════════════════════════════════

TIMELINE OF HINTS FOR STUCK PLAYERS:

0-5 MIN: No hints given (Let them read puzzle)

5+ MIN: 
  └─ Player clicks "HINTS" button
  └─ Enters Admin Password: ADMIN (or ADMIN123)
  └─ Unlocks: TIER 1 HINT (Hard - Explains Cipher Mechanic)

10+ MIN: If still struggling
  └─ Player clicks "HINTS" button again
  └─ Enters Admin Password: ADMIN
  └─ Unlocks: TIER 2 HINT (Medium - Thematic Clue)

15+ MIN: If completely stuck
  └─ Player clicks "HINTS" button again
  └─ Enters Admin Password: ADMIN
  └─ Unlocks: TIER 3 HINT (Easy - Exact Equation)

GAME MASTER OVERRIDE:
  └─ Game Master can enter ADMIN at any time to reveal hints
  └─ Game Master can provide the exact password as a last resort
